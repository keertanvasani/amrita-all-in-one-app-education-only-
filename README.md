AUMS (Amrita University Management System) is a centralized academic management platform designed for Amrita Vishwa Vidyapeetham.
This project proposes a modernized and optimized implementation of AUMS, addressing usability, scalability, and performance limitations in the existing system.

The system provides a single unified interface for managing academic activities such as course registration, assignment submissions, attendance tracking, examination results, fee payments, and library services.
